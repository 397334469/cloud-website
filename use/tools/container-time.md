# 调整容器时间

> 在我们项目维护过程中，可能会遇到需要修改服务器时间，平台的工具集功能就可以满足您的需求了~



在这需要注意，此功能依赖faketime，请在宿主机编译faketime扩展。路径在 `/usr/local/lib/libfaketime.so.1`

## 调整服务时间（菜单栏：工具集->调整时间）

在项目列表中筛选您要修改的项目，点击 `修改时间`，确认之后会重启服务生效。
